# Orca Tabs Plugin

为 Orca Note 提供浏览器式标签页管理功能的强大插件。

> **⚠️ 重要：首次使用请在设置中配置"主页块ID"（默认为1），否则可能出现BUG！**

## 🚀 快速开始

### 安装

1. 下载插件包并解压
2. 将文件夹放置到 Orca Note 插件目录
3. 重启 Orca Note
4. 在设置中配置主页块ID

### 设置主页块ID

1. 选中需要设置为新标签页的块
2. 右键块标 → 选中更多 → 复制块ID
3. 粘贴到插件设置页面中的"主页块ID"中

## 🎯 使用方法

### 基本操作

| 操作 | 方法 |
|------|------|
| 切换标签 | 点击标签页 |
| 关闭标签 | 中键点击标签 |
| 固定标签 | 双击标签 |
| 拖拽排序 | 拖动标签到目标位置 |
| 右键菜单 | 右键标签查看更多操作 |

### 新建标签页

1. **块菜单** - 右键块标选择"在新标签页打开"
2. **Ctrl+点击** - 按住 Ctrl 点击块引用在后台打开
3. **加号按钮** - 点击标签栏的 + 按钮

### 工作区管理

1. 点击顶部工具栏的工作区按钮
2. 选择"保存当前工作区"
3. 输入名称和描述
4. 在列表中快速切换工作区

### 布局切换

- 右键新建标签页的 + 按钮
- 选择水平/垂直布局
- 选择是否固定到顶部

## ⚙️ 配置选项

在 Orca Note 设置中找到"Orca Tabs Plugin"：

- **主页块ID** - 新建标签页的默认块ID（必填）
- **缓存编辑器数量** - 最大标签页数量
- **显示块类型图标** - 是否显示块类型图标
- **启用工作区功能** - 是否启用工作区管理
- **启用最近关闭标签页** - 是否记录最近关闭
- **启用多标签页保存** - 是否允许保存标签组合
- **刷新后恢复聚焦标签页** - 是否自动恢复当前标签
- **启用贴边隐藏功能** - 是否启用贴边隐藏
- **启用气泡模式** - 是否启用气泡模式（仅垂直布局）

### 块属性支持

在块属性中设置：
- `_color` - 标签背景色和文字颜色（type=1）
- `_icon` - 标签图标（type=1）

## ❓ 常见问题

**Q: 标签栏什么时候显示？**  
A: 检测到活跃编辑器块时自动显示。

**Q: 标签数量有限制吗？**  
A: 受"缓存编辑器数量"设置控制。

**Q: 如何固定标签？**  
A: 双击标签切换固定状态。

**Q: 如何关闭标签？**  
A: 中键点击或右键菜单关闭。

**Q: 固定标签可以关闭吗？**  
A: 固定标签需要先取消固定才能关闭。

**Q: 工作区和标签集合的区别？**  
A: 工作区会替换当前所有标签，标签集合只是保存一组标签供添加。
